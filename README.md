# Real-Time Hand Gesture Classification

A computer vision project that classifies hand gestures in real time using webcam input and classical machine learning.

---

## Features
- Webcam-based hand tracking
- MediaPipe landmark extraction
- Random Forest classifier
- Real-time prediction

---

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Scikit-learn

---

## How to Run
```bash
pip install opencv-python mediapipe scikit-learn joblib pandas
python collect_data.py
python train_model.py
python predict_live.py
```

 ## Gestures
-Open hand
